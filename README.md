# little italy

Sitio web de una página para **little italy** — venta de pastas y salsas
proteicas congeladas. La propuesta: comida italiana saludable, simple y lista en
minutos.

## Secciones

- **Hero** con la propuesta de valor y llamados a la acción.
- **Productos**: pastas proteicas, salsas caseras y la ventaja de tenerlas listas en minutos.
- **Cómo funciona**: tres pasos para pedir, guardar y disfrutar.
- **Contacto**: datos directos y un formulario de pedido.

## Tecnología

- [Next.js 16](https://nextjs.org/) (App Router)
- [Tailwind CSS v4](https://tailwindcss.com/)
- TypeScript

## Cómo correrlo localmente

```bash
npm install
npm run dev
```

Abrí [http://localhost:3000](http://localhost:3000) en el navegador.

## Build de producción

```bash
npm run build
npm run start
```
